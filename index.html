Philosophy

The primary aim of this website is to provide strong support for synchronous (live), asynchronous (turn-based), and hybrid go games with both synchronous and asynchronous phases.

Browser Support

Supporting old browsers can quickly become a burden, especially if we want to achieve the kind of high interactivity only modern web technologies provide. Thus, we use html5 and Socket.IO with the flash fall-back disabled. Most browsers support these features, with the notable exception of Internet Explorer, where WebSocket is only supported in version 10.

Testing is done with Google Chrome, which is free to use, and we recommend it.
